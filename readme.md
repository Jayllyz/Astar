# ASTAR

Projet 2A-S2 ESIEA réalisé par @Jayllyz & @Pierrick444

Le projet consiste à utiliser l'algorithme de recherche Astar pour trouver le bon coffre parmi 4 coffres possible sur une map
de 40 x 40.
